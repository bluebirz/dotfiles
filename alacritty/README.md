# [Alacritty](https://alacritty.org)

- install via homebrew
- in case of no process run:
  1. execute `sudo spctl --master-disable`
  1. launch Alacritty again
  1. execute `sudo spctl --master-enable`

> [!NOTE]
> Prefer `ghostty` instead.
